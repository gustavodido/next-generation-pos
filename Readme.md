# Next Generation Point of Sales

## Goal

NDA

## Running

### Docker

Go do docker/
	- ./start.sh - To start the container
	- ./stop.sh - To stop the container

### Web Api

Go to the WebApp/ folder and run `dotnet run`.

### Tests

Go to the WebApi.Tests/ folder and run `dotnet test`.
